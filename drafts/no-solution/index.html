无解

Written by lyumn

某通讯软件（slack）在每次启动载入的时候，会扔出一句名言警句来填补空虚。某天，它告诉我：“Life is not a problem to be solved, but a reality to be experienced（参考翻译：生活不是一个亟待解决的问题，而是一个任由感受的现实。）”。

这是丹麦哲学家齐克果的话。很多人的工作主题，尤其是工程师的工作主题，都是在“解决问题”。

马克思说：“對自然的控制增加會伴隨著異化，也就是人類生活受到人類的發明所控制的狀態。”

电脑很慢，互联网信息不安全等这一类的烦恼很常见，但如果没有电脑，以及没有互联网，那么这一类烦恼就根本不会存在。

但如果没有互联网，信息之间的交流是那么没有效率。如果互联网的出现是解决“沟通效率缓慢”的问题，那么我想现在很多人的工作则是在解决“沟通效率变快”所带来的问题。

既然问题解决不完，为什么在一开始的时候人们会寻求解决问题？

因为生活不是为了消灭问题，而是为了生活本身。如果生活本身美丽令人满意的话，原本的问题是否得到解决是不重要的，原本的目标是否被达成是不重要的。未解决的问题可能会变得不再是问题，未达成的目标可能会变得不再是目标，但生活总是生活。
